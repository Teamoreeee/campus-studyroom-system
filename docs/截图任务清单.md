| API 网关 | http://localhost:8000 | — |
| 接口文档（Swagger） | http://localhost:8000/swagger-ui/index.html | — |
| Nacos 控制台 | http://localhost:8758/nacos | nacos / nacos |
| MySQL（外部连接） | localhost:3307 | root / 123456 |
| Redis（外部连接） | localhost:6380 | 无密码 |

> ⚠️ 注意端口：MySQL 是 **3307**（非 3306）、Redis 是 **6380**（非 6379）、Nacos 是 **8758**（非 8848）——宿主端口为避开本地占用/Windows 保留端口而重映射，容器内部仍是标准端口。
