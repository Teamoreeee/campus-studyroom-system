## 录制建议

### 画面录制
- 使用 OBS Studio 或 Windows 自带 Xbox Game Bar 录制屏幕
- 建议录制 PPT 全屏播放 + 系统实际操作画面
- 关键操作（如 AI 推荐、登录、docker compose ps）可适当放大或加鼠标高亮

### 配音录制
- 使用麦克风或耳机录音，避免环境噪音
- 每段旁白可单独录制，后期拼接
- 语速控制在每分钟 220–250 字

### 后期剪辑
- 添加转场动画，但不宜过多
- 关键页面停留 3–5 秒
- 操作演示部分保留完整操作流程，不要剪辑过快
- 背景音乐可选，音量要小，不要盖过旁白

### 输出规格
- 格式：MP4
- 分辨率：1920×1080
- 帧率：30fps
- 码率：5–8 Mbps
- 文件名：`校园自习室预约系统_演示视频_v1.mp4`
