## 三、不要打包进去的文件/目录

| 项目 | 原因 |
|------|------|
| `frontend/node_modules/` | 体积巨大，老师不会看，且可通过 `npm install` 还原 |
| `backend/**/target/` | Maven 编译产物，可通过 `mvn package` 还原 |
| `logs/` | 运行时日志，非交付物 |
| `C:\temp\campus-studyroom-build/` | 本地临时构建目录 |
| `.git/` | 代码仓库元数据，通常不需要（如老师要求提交 Git 历史则另说） |
| `compile-errors.log` / `init-db-error.log` / `auth.log` | 错误日志 |
| `*.tmp` / `*.pid` / `nohup.out` | 临时文件 |
| `2026-06-*.txt` | 会话导出文件 |
| `ai_chat_*.json` / `ai_rec_*.json` | 本地测试 JSON，非必要 |

## 五、快速打包命令（PowerShell）

在 `campus-studyroom` 目录下执行：

```powershell
# 清理 node_modules 和 target（如未清理）
Remove-Item -Recurse -Force frontend\node_modules
Get-ChildItem -Path backend -Recurse -Filter target | Remove-Item -Recurse -Force

# 打包
Compress-Archive -Path backend,frontend,pom.xml,docker-compose.yml,docker,deploy,k8s,init-db.bat,build-images.sh,start-backend.bat,stop-all.ps1,test,docs,选题申请表.md,评分标准.txt -DestinationPath ..\CampusStudio_校园自习室预约系统_提交材料.zip
```

> 视频文件 `demo-video.mp4` 建议单独上传或放在网盘，避免压缩包过大。
