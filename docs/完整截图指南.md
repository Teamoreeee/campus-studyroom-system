## 一、截图工具推荐

### Windows 自带（最方便）

| 快捷键 | 功能 |
|--------|------|
| `Win + Shift + S` | 区域截图，截完后自动复制到剪贴板 |
| `Win + PrtSc` | 全屏截图，自动保存到 `图片/屏幕截图/` |
| `Alt + PrtSc` | 截取当前活动窗口 |

### 推荐第三方工具

1. **Snipaste**（强烈推荐）
   - 按 `F1` 截图，`F3` 贴图
   - 支持标注、马赛克、文字
   - 截图后按 `Ctrl + S` 保存
   - 官网：https://www.snipaste.com

2. **ShareX**
   - 功能强大，支持滚动截图、GIF、录屏
   - 适合截长网页、终端长输出

3. **QQ / 微信截图**
   - `Ctrl + Alt + A`（QQ）
   - `Alt + A`（微信）
   - 方便快捷，但分辨率可能略低

### 浏览器截网页

- 按 `F12` 打开开发者工具
- `Ctrl + Shift + P` 输入 `full size screenshot` 可截整页
- 适合截 Nacos 控制台、系统长页面
